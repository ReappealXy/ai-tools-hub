## 🖥️ 本地运行

```bash
git clone https://github.com/ReappealXy/ai-tools-hub.git
cd ai-tools-hub
# 用浏览器直接打开 index.html
```

## 🚩 部署到 GitHub Pages

1. 推送代码到 GitHub 仓库
2. 在仓库页面点击 `Settings` → `Pages`
3. 选择 `main` 分支，保存
4. 稍等片刻即可通过 `https://你的用户名.github.io/ai-tools-hub/` 访问
